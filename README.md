# video to slides

Use this script to create png files from video files.
It uses skimage and opencv-python libraries.



	usage: video_to_images.py [-h] [-o OUTPUT_DIRECTORY] video_filename

	positional arguments:
	  video_filename        video filename to load

	options:
	  -h, --help            show this help message and exit
	  -o OUTPUT_DIRECTORY, --output_directory OUTPUT_DIRECTORY
	                        output directory to save png images files extracted from video
